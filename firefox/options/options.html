Personal Access Token Setup

How to create a Personal Access Token:

  1. Go to your GitLab instance → User Settings → Access Tokens
  2. Or use this direct link: your-gitlab-url/-/user_settings/personal_access_tokens
  3. Name: GitLab Notifier (or anything you like)
  4. Set an expiration date (optional but recommended)
  5. Select scope: read_api (minimum required)
  6. Click Create personal access token
  7. Copy the token — you won't see it again!
